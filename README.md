# ML-RGR

Проект представляет собой Streamlit-приложение для анализа датасета автомобилей и инференса моделей машинного обучения. Целевая переменная - `Price(euro)`, то есть цена автомобиля в евро.

Данные берутся из файла `cars.csv`. В качестве признаков используются пробег, объем двигателя, возраст автомобиля, тип топлива, коробка передач, марка и тип кузова. Категориальные признаки представлены one-hot кодированием.

В приложении доступны:

- краткая информация о разработчике;
- описание датасета и этапов предобработки;
- визуализации зависимостей в данных;
- прогноз стоимости автомобиля через ручной ввод параметров или загрузку CSV.

## Модели

В проекте обучены и сохранены следующие модели регрессии:

- `ML1_polynomial_ridge` - Ridge-регрессия с полиномиальными признаками степени 2;
- `ML2_gradient_boosting` - Gradient Boosting Regressor;
- `ML3_catboost` - CatBoostRegressor;
- `ML4_bagging` - BaggingRegressor на основе деревьев решений;
- `ML5_stacking` - StackingRegressor с Ridge, Random Forest и Gradient Boosting;
- `ML6_mlp_neural_network` - полносвязная нейронная сеть MLPRegressor.

Метрики моделей хранятся в `models/metadata.json`. Оценка качества выполнялась на тестовой выборке 20% с фиксированным `random_state=42`.

## Запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Обучение моделей:

```bash
python train_models.py
```

Запуск приложения:

```bash
streamlit run app.py
```
